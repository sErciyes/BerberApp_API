# Screenshot Checklist

LinkedIn, GitHub README veya is gorusmesinde repo gosterirken asagidaki ekranlar en guclu set olur.

## Oncelikli 6 Ekran

1. Login
   - Temiz auth UI
   - Dark mode

2. Register / Barber kaydi
   - Kullanici-Berber ayrimi
   - Uzmanlik alani gorunsun

3. Shops / Harita
   - Dukkan marker'lari
   - Yakin dukkan listesi

4. Randevu olusturma
   - Tarih kartlari
   - 30 dakikalik slot butonlari
   - Sagda randevu ozeti

5. Mesajlasma
   - Konusma listesi
   - Aktif chat paneli

6. Profil / Guvenlik
   - Email ve telefon dogrulama
   - Sifre degistirme alanlari

## Guclu Ek Ekranlar

7. Berber tarafi randevu takvimi
   - Bugun saat saat randevular
   - Ileri tarihli tablo

8. Admin ekrani
   - Berber yonetimi
   - Randevu yonetimi

9. Swagger
   - JWT authorize
   - Endpoint yapisi

## Screenshot Alirken

- Aynı tema ile cekim yap: tercihen dark mode
- Demo veriler anlamli olsun
- Toast/error yerine basarili state yakala
- Sidebar ve sayfa icerigi ayni karede gorunsun
- Tarayici sekmeleri daginik olmasin
- Gereksiz kişisel bilgi gorunmesin
