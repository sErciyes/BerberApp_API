# BerberApp UI

React + Vite + TypeScript ile hazirlanan sade frontend uygulamasidir.

## Kurulum

```bash
npm install
npm run dev
```

Varsayilan API adresi:

```text
http://localhost:5159/api
```

Farkli bir API adresi kullanmak icin `.env` dosyasi olusturun:

```text
VITE_API_BASE_URL=http://localhost:5159/api
```
